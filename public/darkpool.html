Operator Panel

        

          

            Live operator controls for the on-chain settlement anchor. Use this to inspect zkApp roots, inspect the current Merkle-backed private state, build the next private-state witness, and prove the next pending batch.
          

          

            Manual operator actions are gated by the operator admin key. The background operator process still runs independently.
          

          

            
            
          

          

            
            
            
            
          

          

        

      

    


    

      

        

          
On-Chain Funding

          

            

              Real-funds-only mode: fund your wallet on Zeko testnet, then sync balances into ShadowBook.
              Limit order collateral is locked at placement and unlocked on fill, cancel, or expiry.
            

            

            

              
              
            

            

          

        


        

          
Funding Guide

          

            

              1) Acquire testnet tETH/tZEKO/tMINA in wallet.
              2) Connect wallet in Trading tab (or paste wallet above).
              3) Click Sync On-Chain Balance.
              4) Send on-chain token transfer to the vault deposit address (wallet signs tx).
              5) Mint note using the signed deposit tx hash, then paste note into order entry.
              6) Place orders.
