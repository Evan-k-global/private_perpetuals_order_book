# Agent Skills Templates

This folder contains publishable skill templates for protocol app teams.

## Templates
- `protocol-dapp-starter`
- `zeko-da-privacy`
- `shared-liquidity-sdk`
- `trading-ui-composer`
- `prod-runbook-debugger`

## Publish Steps
1. Copy each template folder into your skills repo as-is.
2. Keep folder name equal to `name` in `SKILL.md`.
3. Add optional `agents/openai.yaml` metadata per your distribution channel.
4. Validate trigger descriptions against your internal usage prompts.

## Suggested Versioning
- Tag bundle versions as `skills-vYYYY.MM.DD`.
- Keep each skill backward compatible where possible.
